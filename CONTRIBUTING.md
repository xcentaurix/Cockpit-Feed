# Contributing to Cockpit-Feed Package Feed

Thank you for your interest in contributing to the Cockpit-Feed project!

## Cloning the Repository

To clone the repository, use the following command:

```
git clone https://github.com/YOUR-USERNAME/Cockpit-Feed.git
```

## Adding Upstream Remote

After cloning, add the upstream remote:

```
git remote add upstream https://github.com/dream-alpha/Cockpit-Feed.git
```

## Changing Directories

While navigating within your terminal, make sure to change to the newly cloned directory:

```
cd Cockpit-Feed
```

## Cockpit Series

For contributions related to the Cockpit series plugins, please refer to the respective documentation.

Happy contributing!
