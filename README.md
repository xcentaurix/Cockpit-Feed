# Cockpit-Feed

##  OPKG Package Feed for Cockpit series OA Enigma2 plugins

[![Publish Package Feed](https://github.com/xcentaurix/Cockpit-Feed/actions/workflows/build-packages.yml/badge.svg)](https://github.com/xcentaurix/Cockpit-Feed/actions/workflows/build-packages.yml)

## Feed Installation
To install the feed enter the following command in a box console:

```
echo "src/gz cockpit-all https://xcentaurix.github.io/Cockpit-Feed/packages/all" > /etc/opkg/cockpit-feed-all.conf
```
